3. Определить частицы X, образующиеся в реакциях сильного взаимодействия:
1) π− + p → K− + p + X; 2) K− + p → Ω− + K0 + X; 3) p + → Ξ− + π+ + X.

Исходя из законов сохранения электрического заряда Q, барионного заряда B, странности S и проекции изоспина I3 в этих реакциях определим характеристики частиц X:

1)

π− + p → K− + p + X

Q:

-1 + 1 -1 + 1 + QX

QX = 0

B:

0 + 1   0 + 1 + BX

BX = 0

S:

0 + 0 -1 + 0 + SX

SX = 1

I3:

- 1 + 1/2 -1/2 + 1/2 + (I3)X

(I3)X = -1/2

Этот набор квантовых чисел соответствует K0-мезону.

2)

K− + p → Ω− + K0 + X

Q:

-1 + 1 -1 + 0 + QX

QX = 1

B:

0 + 1   1 + 0 + BX

BX = 0

S:

-1 + 0   -3 + 1 + SX

SX = 1

I3:

-1/2 + 1/2 0 - 1/2 + (I3)X

(I3)X = 1/2

Этот набор квантовых чисел соответствует K+-мезону.

3)

p + → Ξ− + π+ + X

Q:

1 - 1 -1 + 1 + QX

QX = 0

B:

1 -  1 1 + 0 + BX

BX = - 1

S:

0 + 0 -2 + 0 + SX

SX = 2

I3:

1/2 -  1/2 -1/2 + 1 + (I3)X

(I3)X = - 1/2

Этот набор квантовых чисел соответствует Ξ0-гиперону.
