56. Показать, что в реакции неупругого рассеяния дейтронов на ядре 10B, идущей за счёт сильного взаимодействия, невозможно возбуждение уровней этого ядра с изоспином I = 1.

Реакция имеет вид d + 10B → d + 10B*. Дейтрон и 10B это ядра с N = Z. Поэтому для них
I3 = |N − Z|/2 = 0, и изоспин основного состояния Igs, определяемый правилом Igs = |I3|, для каждого из этих ядер также равен нулю:Igs = 0. Отсюда получаем, что сохранение изоспина в рассматриваемом процессе возможно лишь, если изоспин образующегося возбуждённого состояния ядра 10B* также равен нулю: I(10B*) = 0.
